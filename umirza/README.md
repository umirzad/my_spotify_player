# my-spotify-player (Frontend)

Bu proje `Vue 3` ve `Vue CLI` ile gelistirilmis bir Spotify benzeri arayuzdur.

## Kurulum

```bash
npm install
```

## Gelistirme

```bash
npm run serve
```

## Build

```bash
npm run build
```

## Ortam Degiskeni

`.env` dosyasina su degiskeni ekleyin:

```bash
VUE_APP_API_BASE_URL=http://localhost:3000
```
